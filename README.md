# wow_discord_news
A small project to redirect the news of the WoW guild to Discord  
A project still in development

### Requirements
* [discord-webhook 0.2.0](https://pypi.org/project/discord-webhook/ )
* [SQLAlchemy](https://www.sqlalchemy.org/)
